# Проект: Mesto

## Содержание:

Study project "Mesto" is aimed at practicing adaptive layout, working with JS, form validation and setting up server interaction

* CSS (GRID, flex, @media-querries)
* JavaScript
* Figma
* BEM
* OOP
* VSCode
* Webpack
* API (server integration)
